
# awesome-pharmacy-fontend

## Project setup
```
npm install
```

```
npm run serve
```

打开```http://localhost:8088/```即可


#### 项目介绍
仿照阿里健康大药房实现的一个项目，后台使用php，前台使用vue全家桶。
此处仅提供前台代码，后台代码可以自行实现。仅需在vue.config.js文件下修改接口代理的路径为自己后台服务器路径即可。
